pwd para saber donde estoy ubicado  
ls lista de archivos  
cd carpeta para ubicar en el explorador y entrar en la carpeta   
mkdir crear carpeta   
ls -al para verificar si se creo la carpeta   
git config --list para verificar   
git branch -M main cambiar de master a main  
git config --global initdefaultbranchmain cambiar todo   
touch crear carpeta   
vim para editar el  archivo creado en touch  
para entrar se usa i   
para salir del vim esc y guardar :x  
q para borrar sin guardar   
cat para verificar que se escribio   
git add como una prueba   
git status verificar   
git rm --cached para sacarlo del stage   
git commit -m "" para subir el commit
